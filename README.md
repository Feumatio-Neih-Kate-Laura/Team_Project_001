# Personal Assistant and Task Tracker App

This app is designed to help users organize their daily lives through automated scheduling, task tracking, and a built-in voice assistant.

---

## 📝 Functional Requirements

The following requirements define what the system shall do:

### 1. User Management
* **FR-01: Sign up and Login** – Users can create an account and log in securely.
* **FR-08: User Profile** – Users can create and manage their personal profiles and settings.
* **FR-09: Account Verification** – The system will allow users to verify their accounts for security.

### 2. Task & Timetable Management
* **FR-02: Daily Task Entry** – Users can enter all their daily tasks into the system.
* **FR-06: Timetable Creation** – The app helps manage the user's timetable and can automatically create one based on the tasks entered.
* **FR-07: User Agenda** – Provide a clear agenda view so users can see their full schedule.

### 3. Assistant & Tracking Features
* **FR-03: Notifications** – Send alerts to keep users informed and make sure they remain on time for their tasks.
* **FR-04: Voice Assistant** – A built-in voice assistant to help users carry out their tasks hands-free.
* **FR-05: Task Records** – The system keeps a record of all tasks and how the user carries them out to track progress.

---

## 🚀 How it Works

1.  **Input:** You list your tasks for the day.
2.  **Organize:** The app builds a timetable for you so you don't have to plan it manually.
3.  **Assist:** The voice assistant and notifications keep you moving through your list.
4.  **Record:** At the end of the day, you can see a record of everything you accomplished.
